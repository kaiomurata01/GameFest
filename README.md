Este projeto é um site de evento de jogos moderno e minimalista, desenvolvido para o GAMERFEST 2026 (evento inventado). 
Ele foi estruturado para ser altamente responsivo e visualmente impactante.

Principais Funcionalidades:
Design Minimalista Neon: Estética focada em tons de preto profundo com detalhes em ciano e roxo neon.

Cronograma Interativo: Grade completa para os 4 dias de evento (30/10 a 02/11) com sistema de abas e suporte a múltiplos palcos.

Detalhes Dinâmicos: Sistema de accordion (clique para expandir) que revela convidados famosos e detalhes de cada atração.

Seção "Sobre": Apresentação do evento com grade de estatísticas.

Localização e Mapas: Integração com Google Maps e mapa interno ilustrativo do pavilhão.

Página de Vendas: Estrutura dedicada para a compra de ingressos e passaportes.

Tecnologias Utilizadas:
HTML5 (Semântico)

CSS3 (Variáveis, Flexbox, CSS Grid)

JavaScript (Manipulação de DOM e lógica de filtros)
